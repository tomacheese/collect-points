## 目的

全20ゲームについて、実際に Docker で実行してポイントが獲得できることを確認する。

## 確認方法

1. `data/config.json` を確認して ECNavi のログイン情報があることを確認
2. Docker コンテナで個別ゲームを実行
3. ログで実行前後のポイントを確認
4. エラーがあれば詳細を記録
5. すべての結果を記録

## 確認手順

### 前提条件

```bash
# 設定ファイルが存在するか確認
ls -la data/config.json

# Docker イメージをビルド
docker compose build
```

### 個別実行方法

各ゲームを個別に実行する方法を調査・確認する必要がある。

#### 方法1: 環境変数で制御

```bash
# 特定のゲームのみ実行できるか確認
ECNAVI_ONLY=chinju pnpm dev
```

#### 方法2: コード一時修正

`src/providers/ecnavi/index.ts` で特定のゲームのみ実行するように一時修正

```typescript
// 他のゲームをコメントアウトまたは条件分岐
```

#### 方法3: ログから確認

全ゲーム実行後、ログから各ゲームのポイント変化を確認

### 確認項目

各ゲーム実行時に確認する項目：

1. **エラーの有無**: エラーログがないか
2. **ポイント変化**: ログに記録されたポイントの増減
3. **実行時間**: タイムアウトしていないか
4. **スクリーンショット**: エラー時のスクリーンショットが保存されているか
